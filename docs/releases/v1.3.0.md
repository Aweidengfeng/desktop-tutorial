# v1.3.0 — 2026-05-11

> **一句话亮点：** 面向移动端提审与生产部署的集中收口版本，支付、合规、上架资料与稳定性都进入可交付状态。

## 🌟 主要亮点
- 移动端提审所需的图标、截图、权限说明、元数据和提交流程已基本齐备。
- 支付能力从测试集成走向生产切换准备，便于正式上线前完成最后一步配置。
- 法律页面、GDPR 横幅、PII 加密与日志脱敏让合规表达更完整。
- 部署流水线、冒烟测试与发布前检查补齐后，运维上线更可控。
- 登录链路与连锁 500 错误被集中修复，整体稳定性显著提升。

## 🆕 新功能
- 新增 Stripe 支付、多货币显示与投资人看板真实数据。
- 新增 App 图标、启动屏、截图模板、提交 runbook 和一键图标生成工具。
- 新增 iOS / Android CI 打包、Fastlane 自动化、Universal Links / App Links 与商店元数据。
- 新增 Railway 生产部署流水线、冒烟测试和发布前自检能力。

## 🔧 改进与修复
- 修复地图路由冲突、Capacitor 版本缺失 GDPR 横幅、法律页错误链接和环境变量文档缺项。
- 修复代码完整性审计发现的脚本映射、Stripe webhook、Universal Links 与横幅问题。
- 修复随机 IV PII 加密导致的登录失败、重复校验失效和连锁 API 500 问题。
- 完成品牌统一与部署清单收口，让对外材料和产品命名保持一致。

## 🛡️ 安全与合规
- 上线 AES-256-GCM PII 字段加密，并进一步修复等值查询与登录链路的一致性问题。
- GDPR 横幅、法律页面和数据权利入口得到补齐，审核员可更清楚看到合规说明。
- 短信、邮件等日志中的手机号、邮箱与验证码已完成脱敏处理，减少敏感信息泄露风险。
- 支付正式上线前加入 Live Mode 守卫，避免生产环境误用测试密钥。

## 📱 平台支持

| 平台 | 状态 | 说明 |
|---|---|---|
| iOS | 提审就绪 | 已补齐图标、权限、打包与提交资料 |
| Android | 提审就绪 | 已补齐图标、权限、AAB 构建与商店资料 |
| Web | 已支持 | 保持完整功能可用 |
| PWA | 已支持 | 图标、缓存与安装体验同步完善 |

## ⬆️ 升级说明
- 对普通用户无额外迁移步骤。
- 对运维团队，建议在正式上线前补齐 Stripe Live 密钥、Universal Links 占位信息、移动签名证书与生产环境变量。
