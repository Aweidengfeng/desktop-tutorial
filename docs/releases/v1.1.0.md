# v1.1.0 — 2026-05-02

> **一句话亮点：** 让“发现内容”和“收到提醒”都更及时，用户从浏览走向持续互动。

## 🌟 主要亮点
- 搜索入口更好用，用户更容易找到山峰、向导、俱乐部和社区内容。
- 重要动作开始具备邮件提醒能力，减少遗漏订单和认证进度的可能。
- Web Push 让回访提醒更自然，平台互动不再只依赖用户主动打开页面。

## 🆕 新功能
- 新增统一搜索与搜索建议，方便快速查找路线、山峰和社区内容。
- 新增邮件通知能力，可覆盖预订确认、认证结果与邮箱验证码等场景。
- 新增 Web Push 订阅与通知基础设施，支持更及时的消息触达。

## 🔧 改进与修复
- 搜索体验同时兼容 PostgreSQL 与 SQLite，两种环境都能保持可用。
- 通知能力采用渐进启用方式，未配置邮件或推送密钥时也不会影响主流程。

## 🛡️ 安全与合规
- 延续 v1.0.0 的 GDPR 数据导出与删除基础能力。
- 新增通知能力以用户主动触发或订阅为前提，不默认扩大数据采集范围。

## 📱 平台支持

| 平台 | 状态 | 说明 |
|---|---|---|
| iOS | 暂未单独发布 | 本版本仍以 Web 使用为主 |
| Android | 暂未单独发布 | 本版本仍以 Web 使用为主 |
| Web | 已支持 | 搜索与通知能力已可用 |
| PWA | 已支持 | 可接收安装后的 Web Push 提醒 |

## ⬆️ 升级说明
- 无 breaking change。
- 如需启用邮件或推送，请按环境变量文档补齐 SMTP / VAPID 配置。
