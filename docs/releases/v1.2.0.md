# v1.2.0 — 2026-05-02

> **一句话亮点：** 把智能陪练、支付底座和运营视图放进同一个版本，平台开始具备更完整的商业化与运营能力。

## 🌟 主要亮点
- AI Coach 让新手和进阶用户都能更容易获得训练建议。
- 支付能力从单点实现走向统一底座，为后续不同支付方式留出空间。
- 管理后台第一次具备聚合统计视图，方便团队观察平台增长情况。
- CI 回归问题在同日修复，保证新能力上线时不过度牺牲稳定性。

## 🆕 新功能
- 新增 AI Coach，支持对训练、装备和行程准备给出建议。
- 新增支付网关抽象层，为多种支付方式提供统一接入方式。
- 新增管理后台统计接口，可查看用户、内容、订单和营收等概览数据。

## 🔧 改进与修复
- 修复测试环境缺少新路由的问题，避免关键接口在 CI 中误报失败。
- 提高管理统计和 AI 上下文查询的容错性，测试环境更稳定。

## 🛡️ 安全与合规
- 管理统计接口要求管理员身份访问，降低后台数据被误用的风险。
- 支付能力以抽象层方式接入，便于在不同环境下逐步配置与验收。

## 📱 平台支持

| 平台 | 状态 | 说明 |
|---|---|---|
| iOS | 暂未单独发布 | 仍以 Web 形态访问为主 |
| Android | 暂未单独发布 | 仍以 Web 形态访问为主 |
| Web | 已支持 | AI Coach 与管理能力已上线 |
| PWA | 已支持 | 延续 Web 端安装体验 |

## ⬆️ 升级说明
- 无 breaking change。
- 如需启用在线 AI 或真实支付，请补齐对应服务密钥；未配置时仍可使用降级能力。
