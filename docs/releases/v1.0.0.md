# v1.0.0 — 2026-04-30

> **一句话亮点：** SummitLink 首个对外可发布版本，完成了社区、路线、轨迹、安全与运维的基础闭环。

## 🌟 主要亮点
- 用户已经可以浏览山峰、记录轨迹、参与社区，并形成完整的户外社交体验。
- 关键使用场景开始支持弱网与离线兜底，减少户外信号不稳定带来的中断感。
- 平台具备基础的安全与合规能力，包括账户删除、数据导出和内容审核。
- 首发版本同时补齐了监控、部署和性能验证，让对外演示更稳定。

## 🆕 新功能
- 支持向导和俱乐部在被拒后重新提交申请，减少流程卡住的情况。
- 支持图片统一管理、轨迹断点续传、PWA 安装与基础离线访问。
- 支持高德 / Mapbox 双地图体验，适应不同地区用户的使用习惯。
- 支持多语言基础框架、深色模式和更现代的前端构建流程。

## 🔧 改进与修复
- 增补并发下单、JWT 鉴权和性能基准测试，提升首发质量可见性。
- 引入多节点部署、OSS / CDN 与健康检查机制，改善上线稳定性。
- README、部署文档和发布材料同步完善，便于团队交接和对外演示。

## 🛡️ 安全与合规
- 提供 GDPR 数据导出与账号删除能力，方便用户管理自己的数据。
- 接入图片内容安全能力，减少违规上传风险。
- 引入安全验收自动化与 Sentry 监控，帮助团队更快发现问题。

## 📱 平台支持

| 平台 | 状态 | 说明 |
|---|---|---|
| iOS | 暂未单独发布 | 首发阶段以 Web 体验为主 |

| Web | 已支持 | 作为对外首发渠道 |
| PWA | 已支持 | 支持安装与基础离线访问 |
