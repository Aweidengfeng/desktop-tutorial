# AlpineLink 统一错误码说明

## 统一响应格式

**成功时：**
```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

**失败时：**
```json
{
  "code": 1003,
  "message": "未登录或Token无效",
  "data": null
}
```

---

## 错误码列表

### 通用错误（1xxx）

| 错误码 | HTTP 状态码 | 说明 |
|--------|------------|------|
| 1001   | 400        | 参数缺失 |
| 1002   | 400        | 参数格式错误 |
| 1003   | 401        | 未登录或 Token 无效 |
| 1004   | 403        | 权限不足 |
| 1005   | 404        | 资源不存在 |
| 1006   | 429        | 请求过于频繁 |

### 认证模块错误（2xxx）

| 错误码 | HTTP 状态码 | 说明 |
|--------|------------|------|
| 2001   | 400        | 验证码已发送，请等待 60 秒 |
| 2002   | 400        | 验证码错误或已过期 |
| 2003   | 403        | 账号已被封禁 |

### 服务器错误（5xxx）

| 错误码 | HTTP 状态码 | 说明 |
|--------|------------|------|
| 5000   | 500        | 服务器内部错误 |

## 使用示例

### 请求示例

```http
POST /api/auth/login
Content-Type: application/json

{
  "phone": "13800138000",
  "code": "123456"
}
```

### 成功响应

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "user": {
      "id": 1,
      "username": "用户13800138000",
      "phone": "13800138000",
      "role": "user"
    }
  }
}
```

### 错误响应示例

**验证码错误：**
```json
{
  "code": 2002,
  "message": "验证码错误或已过期",
  "data": null
}
```

**未登录访问受保护接口：**
```json
{
  "code": 1003,
  "message": "未登录或Token无效",
  "data": null
}
```

**权限不足（普通用户访问管理员接口）：**
```json
{
  "code": 1004,
  "message": "权限不足",
  "data": null
}
```
